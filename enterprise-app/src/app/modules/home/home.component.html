Gestión de Empresas

Este módulo permite registrar empresas (con validación de RNC en DGII) y administrar el catálogo mediante operaciones CRUD contra el WebAPI (.NET). Usa el menú lateral para navegar.

Requerimientos de la prueba

check_circle
Formulario “Crear empresa”
RNC (búsqueda DGII), Nombre, Nombre comercial, Categoría, Esquema de pago, Estado, Actividad económica, Rama gubernamental.
check_circle
CRUD completo
Operaciones contra WebAPI (.NET) usando procedimientos almacenados en SQL Server.
check_circle
Paginación en frontend
Listado con MatTable + MatPaginator.

Estructura del Frontend

folder
modules/companies
models (api/model), adapters, services, pages (create/manage).
 view_sidebar
layout
Barra lateral fija + toolbar; el contenido se renderiza en el centro.

Notas técnicas

info
Angular Material
Theme predefinido, densidad ligera y componentes standalone.
 dns
API .NET
Endpoints: /api/companies (CRUD) y /api/dgii/{rnc} para búsqueda RNC.
security
Validaciones
